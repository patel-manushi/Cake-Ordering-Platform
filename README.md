# Cake-Ordering-Platform
<p align="justify">
The Cake Ordering Platform is a web-based application developed using PHP, HTML, CSS, JavaScript, and MySQL designed to provide a smooth and efficient online cake and chocolate ordering experience. The platform enables users to browse products with detailed information such as images, prices,names and available quantities. It includes secure user registration and login functionality, along with a fully functional shopping cart that allows users to add items, update quantities, remove products, view item-wise totals, and automatically calculate the grand total before checkout. All application data is securely stored and managed using a MySQL database, ensuring data accuracy, reliability, and easy maintenance.
</p>
